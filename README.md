## How To Use

To clone and run this portfolio locally, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/).

```bash
# Clone this repository
$ git clone https://github.com/Suryaraja1586/suryaraja-dev

# Go into the repository
$ cd src

# Install dependencies
$ npm install

# Run the app
$ npm start
```
